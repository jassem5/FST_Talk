# 🚗 DonGfeng App

Welcome to the **DonGfeng App**, a premium, high-end Flutter application designed for a modern automotive experience. This app features sleek animations, climate control, and real-time vehicle status monitoring.

![Dongfeng Logo](assets/images/dongfeng_logo.png)

## ✨ Features

- **🏆 Premium Splash Screen**: A minimalist and professional entry experience with the Dongfeng branding and "Driving the Future" slogan.
- **🌡️ Smart Climate Control**: Dynamic temperature adjustment with interactive %+1% and %-1% controls (ranging from 16°C to 32°C).
- **🔋 Battery Monitoring**: Real-time visualization of the vehicle's battery status and remaining charge.
- **🔒 Security Management**: Control and monitor door locks, trunk, and bonnet status with a simple tap.
- **🛞 Tyre Pressure System**: Monitor individual tyre PSI levels with dedicated status cards and animations.
- **🎨 Modern UI**: Built with a dark-themed aesthetic, smooth transitions, and high-quality SVG iconography.

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Latest stable version recommended)
- [Dart SDK](https://dart.dev/get-started/sdk)
- A physical device or emulator (Android/iOS)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jassem5/FST_Talk.git
   cd FST_Talk
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Run the app**:
   ```bash
   flutter run
   ```

## 📂 Project Structure

- `lib/main.dart`: Entry point of the application.
- `lib/home_controller.dart`: Central state management for vehicle controls and animations.
- `lib/screens/`: Contains all UI screens including the Home and Splash screen.
- `lib/screens/components/`: Reusable widgets like temperature controls, battery status, and tyre cards.
- `assets/`: Icons and images used throughout the application.
